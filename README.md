## ✨ Features & Highlights

The portfolio is a robust React-based Single Page Application (SPA), meticulously crafted with **Next.js 16** and **TypeScript**.

- 🎨 **Modern & Premium Design**: Features a beautiful, dark-themed UI with engaging gradients, carefully designed to highlight content and provide an excellent UX.
- ⚡ **Performance First**: Optimized loading times and SEO, leveraging statically generated pages to ensure a buttery smooth experience.
- 📱 **Fully Responsive Layout**: Built with a mobile-first approach, ensuring seamless adaptability across phones, tablets, and desktop displays.
- ✨ **Dynamic Animations & Interactions**: Integrated with [Framer Motion](https://www.framer.com/motion/) for fluid page transitions, scrolling effects, and a dynamic typing cycle in the hero section.
- 📊 **Analytics Integration**: Set up with Vercel Analytics to track performance and user engagement.

---

## 💻 Tech Stack

- **Framework**: [Next.js (v16)](https://nextjs.org/)
- **Core**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS (v4)](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Optimized [Poppins](https://fonts.google.com/specimen/Poppins) font via `next/font`

## 🚀 Getting Started Locally

Interested in exploring the codebase or running it locally? Follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Tauqeer7Khan/Tauqeer_Khan_Portfolio.git
   cd Tauqeer_Khan_Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Experience the app:**
   Open [http://localhost:3000](http://localhost:3000) in your browser. Edit `app/page.tsx` to see your changes applied dynamically.
